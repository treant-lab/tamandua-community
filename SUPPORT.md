# Support

Use the right channel so the request can be handled safely.

## Public Support

- Discord: fast setup help, onboarding, support triage, and contributor coordination.
- GitHub Discussions: searchable Q&A, benchmark interpretation, and design questions.
- GitHub Issues: reproducible bugs and actionable work items.

Discord invite: https://discord.gg/52Duv5GVDF

## Component Bugs

Open component-specific bugs in the owning repository:

- `tamandua-server`
- `tamandua-agent`
- `tamandua-browser-extension`
- `tamandua-ctl`
- `tamandua-core`
- `tamandua-detection-validation`
- `tamandua-gui`

Use this repository for cross-project proposals and community process.

## Security Reports

Do not report vulnerabilities publicly. Follow [SECURITY.md](SECURITY.md).

## Before Sharing Logs

Remove:

- credentials;
- private keys;
- tokens;
- customer names;
- hostnames and usernames where sensitive;
- private IPs when not required;
- raw endpoint telemetry that identifies people or organizations.
