# Detection Ideas

Use this directory for public-safe detection design notes.

Do include:

- behavior being detected;
- required telemetry;
- expected rule logic;
- benign cases;
- false-positive risks;
- ATT&CK mapping where useful;
- safe reproduction notes.

Do not include:

- live malware samples;
- exploit payloads;
- private incident logs;
- customer data;
- credentials or secrets.
